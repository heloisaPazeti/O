# Sorting Algorithm

### Bubble Sort

Asymptotic Analysis:
- Best Case: $O(n)$
- Worse Case: $O(n^2)$

### Insertion Sort

Asymptotic Analysis:
- Best Case: $O(n)$
- Worse Case: $O(n^2)$

### Selection Sort

Asymptotic Analysis:
- Best Case: $O(n²)$
- Worse Case: $O(n²)$

### Merge Sort

Asymptotic Analysis:
- Best Case: $O(n\log(n))$
- Worse Case: $O(n\log(n))$

### Quick Sort

Asymptotic Analysis:
- Best Case: $O(n\log(n))$
- Worse Case: $O(n²)$

### Heap Sort

Asymptotic Analysis:
- Best Case: $O(n\log(n))$
- Worse Case: $O(n\log(n))$

### Shell Sort

Asymptotic Analysis:
- Best Case: $O(n\log(n))$
- Worse Case: $O(n²)$
